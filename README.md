- Processes bounties
- Runs AI queries
- Earns from all services

- Routes traffic only
- Doesn't store data
- Lower liability
- Still earns fees

### Exit Node
- Final hop in proxy chain
- Higher risk, higher reward
- Optional configuration
- Legal protections vary
